# Проект для практики спецификаций

Этот проект является специализированной средой для практики технического письма, проектирования спецификаций и рабочих процессов разработки с использованием ИИ.

## Цели
- Овладеть искусством написания четких и понятных технических спецификаций.
- Практиковаться в создании проектов на основе спецификаций с помощью ИИ-агентов.
- Изучать современный веб-дизайн и премиальную эстетику.

## Структура
- `specs/`: Технические спецификации и требования к проекту.
- `src/`: Исходный код и реализации.
- `agents.md`: Инструкции и роли, специфичные для агента.
