# Спецификация функции: Каталог курсов (Course Catalog)

## 1. Поведение функции
Список образовательных программ для глубокой проработки проблем со здоровьем.
*   **Тегирование:** Классификация курсов по сложности (Базовый, Продвинутый) и направлению (Детокс, Гормоны, Сон).
*   **Превью:** Просмотр короткого ознакомительного видео или программы курса перед покупкой.

## 2. Данные пользователя
*   **Карточка курса:** Название, автор, длительность в часах/неделях, количество уроков.
*   **Цена:** Текущая цена и старая цена (если есть скидка).
*   **Результаты:** Краткое описание того, что получит пользователь после прохождения.

## 3. Исключительные случаи и Алгоритмы восстановления

### 3.1. Курс снят с продажи или обновляется
*   **Ситуация:** Ссылка ведет на курс, который временно недоступен для покупки.
*   **Инструкция по исправлению:**
    *   Показать сообщение: "Данный курс обновляется и будет доступен скоро".
    *   Предложить подписаться на уведомление о запуске нового потока.
    *   Показать альтернативные курсы по этой же теме.

### 3.2. Ошибка воспроизведения промо-видео
*   **Ситуация:** Видео-превью не загружается из-за внешнего сбоя (например, Vimeo/YouTube).
*   **Инструкция по исправлению:**
    *   Скрыть видеоплеер.
    *   Заменить его на качественное текстовое описание программы курса и PDF-брошюру.
