# Спецификация функции: Каналы связи (Communication Channels)

## 1. Поведение функции
Обеспечение связи пользователя с администрацией или консультантами.
*   **Виджеты мессенджеров:** Плавающая кнопка с выбором WhatsApp или Telegram.
*   **Форма заявки / Почта:** Возможность отправить подробное описание своих проблем со здоровьем для получения индивидуальной рекомендации по выбору курса.
*   **Форма заявки:** Поля для ввода вопроса и контактных данных.

## 2. Данные пользователя
*   **Статус:** "Мы онлайн" / "Ответим в течение часа".
*   **Поля формы:** Имя, телефон (маска ввода), текст сообщения.

## 3. Исключительные случаи и Алгоритмы восстановления

### 3.1. Ошибка отправки формы (Технический сбой)
*   **Ситуация:** Пользователь заполнил форму, но сервер не принял данные.
*   **Инструкция по исправлению:**
    *   Сохранить введенный текст в локальном хранилище браузера (Local Storage).
    *   Показать сообщение: "Упс! Не удалось отправить сообщение. Мы сохранили ваш текст — попробуйте нажать кнопку отправки еще раз через минуту".
    *   Предложить прямые ссылки на WhatsApp как альтернативу.

### 3.2. Некорректный формат данных
*   **Ситуация:** Введен неверный номер телефона или пустой Email.
*   **Инструкция по исправлению:**
    *   Мгновенная валидация: подсветить красным неверное поле.
    *   Подсказка: "Пожалуйста, введите номер в формате +7 (XXX) XXX-XX-XX".
