# Спецификация функции: Каталог товаров (Product Catalog)

## 1. Поведение функции
Отображение полного перечня доступных витаминов и БАДов с возможностью фильтрации.
*   **Вид:** Сетка (Grid) или Список (List).
*   **Микро-теги (Бейджи):** Обязательное наличие цветных поясняющих тегов поверх фото товара (например, "Для сна", "Энергия", "Усвояемость 99%"), чтобы помочь нерешительным покупателям быстро понять разницу между похожими препаратами (например, разными формами магния).
*   **Фильтрация:** Боковая панель с параметрами (цена, цель, бренд).
*   **Сортировка:** Популярность, цена (возр/убыв), новинки.
*   **Пагинация:** Загрузка дополнительных товаров кнопкой "Показать еще" или автоматическая прокрутка.

## 2. Данные пользователя
*   **Боковая панель фильтров:** Чекбоксы, ползунки цен.
*   **Мини-карточки товаров:** Фото с цветными бейджами-подсказками, название, краткое назначение, цена, кнопка быстрого добавления в корзину.
*   **Активные фильтры:** Теги выбранных параметров с кнопкой "Сбросить все".

## 3. Исключительные случаи и Алгоритмы восстановления

### 3.1. Ошибка загрузки каталога (Серверная ошибка)
*   **Ситуация:** Сбой на стороне сервера, данные о товарах не получены.
*   **Инструкция по исправлению:**
    *   Показать сообщение: "Не удалось загрузить каталог. Мы уже работаем над исправлением".
    *   Кнопка "Попробовать еще раз" для повторного запроса.
    *   Предложить связаться через WhatsApp для ручного подбора.

### 3.2. Пустая категория после фильтрации
*   **Ситуация:** Пользователь выбрал слишком жесткие фильтры, под которые не подходит ни один товар.
*   **Инструкция по исправлению:**
    *   Показать сообщение: "Товаров с такими параметрами пока нет".
    *   Кнопка "Сбросить фильтры".
    *   Блок "Рекомендуемые товары" (бестселлеры).

### 3.3. Медленный интернет (Slow connection)
*   **Ситуация:** Картинки товаров загружаются слишком долго.
*   **Инструкция по исправлению:**
    *   Использование скелетон-экранов (Skeletons) вместо пустых мест.
    *   Загрузка изображений в низком разрешении (Blur-up) с постепенным уточнением.
