# ИИ-Навык (AI Skill): Отступы и Сетки (Design Spacing & Layout)

Для достижения премиального вида (Apple-style / Clean UI), ИИ обязан строго соблюдать правила отступов (Spacing). "Воздух" (Negative space) — главный инструмент премиального дизайна.

## 1. Базовая Сетка (8px Grid)
Все отступы (margin, padding), размеры иконок и высоты блоков должны быть кратны 8 пикселям (в Tailwind это стандартные классы `p-2`, `m-4` и т.д.).
*   `8px` (Tailwind: `2`) — микро-отступы (между иконкой и текстом).
*   `16px` (Tailwind: `4`) — стандартный внутренний отступ для небольших карточек.
*   `24px` (Tailwind: `6`) — отступ между связанными элементами или стандартный padding.
*   `32px` (Tailwind: `8`) — отступ между секциями.
*   `64px` (Tailwind: `16`) — большие отступы между крупными логическими блоками на странице.

## 2. Использование "Воздуха" (White Space)
*   Никогда не прижимать текст вплотную к краям экрана или краям карточки.
*   Минимальный боковой padding для всей страницы на мобильных устройствах: `16px` (`px-4`). На ПК: `max-width: 1200px` по центру (`mx-auto px-8`).
*   Если сомневаетесь, сделайте отступ больше, а не меньше. Премиальный дизайн "дышит".

## 3. Выравнивание (Alignment)
*   Использовать Flexbox (`flex`) и Grid (`grid`) для идеального выравнивания элементов по вертикали и горизонтали.
*   Иконки и текст рядом с ними всегда должны быть отцентрированы по вертикали (`items-center`).
